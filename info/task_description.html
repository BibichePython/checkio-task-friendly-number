Numbers for IT should look prettier. So let's write code for that.

You should write a function for converting a number with the several rules. For start you need cut the number with a given base (default 1000). The number should be a coefficient with letters designation of the power. The coefficient is a real number with "decimals" digits after point (default value of decimals is 0). Also for function given a list of powers designations (default ['', 'k', 'M', 'G', 'T', 'P', 'E', 'Z', 'Y']). If you are given suffix, then it must be added in the end. For the coefficient is using rounding down rule (5.6⇒5, -5.6⇒-5) if decimal=0, else using the standard rounding.

Let's look at examples. It will be simpler.

  • n=102
    result: "102", because base is default 1000 and 102 is lower this base.
  • n=10240
    result: "10k", because base is default 1000.
  • n=12341234, decimals=1
    result: "12.3M".
  • n=12461, decimals=1
    result: "12.5k".
  • n=1024000000, base=1024, suffix='iB'
    result: '976MiB'.
  • n=-150, base=100, powers=['', 'd', 'D']
    result: '-1d'.
  • n=-155, base=100, decimals=1, powers=['', 'd', 'D']
    result: '-1.6d'.
